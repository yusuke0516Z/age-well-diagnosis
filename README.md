# Age-Well 診断システム

高齢者の well-being を7つのカテゴリーで診断し、個別化されたフィードバックを提供するWebアプリケーション。

## 📋 機能

### 一般ユーザー向け
- 📝 **診断テスト**: 7カテゴリー × 5問 = 35問の質問
- 📊 **結果表示**: レベル判定（Lv.1〜5）と詳細分析
- 🎨 **可視化**: レーダーチャートとカテゴリー別スコア
- 📄 **PDF出力**: 診断結果のダウンロード

### 管理者向け
- 👥 **回答一覧**: 全診断結果の閲覧
- 📈 **ダッシュボード**: 統計とグラフ
- 🔍 **検索・フィルター**: レベル、ステータス別
- 💾 **CSV エクスポート**: データの一括ダウンロード

## 🚀 クイックスタート

### 1. セキュリティ設定（必須）

⚠️ **この手順を完了しないと管理画面にログインできません**

```bash
# 1. setup-security.html をブラウザで開く
open setup-security.html

# 2. 管理者情報を入力
# - メールアドレス
# - 強力なパスワード（12文字以上推奨）
# - GAS URL（オプション）

# 3. 「config.js 生成」→「コピー」

# 4. config.js に貼り付けて保存
```

### 2. Google Apps Script のセットアップ（オプション）

診断結果をGoogle スプレッドシートに保存する場合:

1. [Google Apps Script](https://script.google.com/) で新規プロジェクト作成
2. 以下のコードを貼り付け（サンプルは別途提供）
3. 「デプロイ」→「新しいデプロイ」→「ウェブアプリ」
4. アクセス: 「自分のみ」に設定（セキュリティのため）
5. デプロイURLを `config.js` の `GAS_URL` に設定

### 3. ローカルで起動

```bash
# シンプルなHTTPサーバーで起動
python3 -m http.server 8000

# または Node.js の場合
npx http-server -p 8000

# ブラウザで開く
open http://localhost:8000
```

### 4. 管理画面へのアクセス

1. トップページ右上の「管理画面」をクリック
2. setup-security.html で設定したメールアドレスとパスワードでログイン

## 🔒 セキュリティ機能

### 実装済み
- ✅ **SHA-256 ハッシュ認証**: パスワードを平文で保存しない
- ✅ **ログイン試行制限**: 5回失敗で15分間ロック
- ✅ **セッション管理**: 1時間の自動タイムアウト
- ✅ **XSS対策**: すべての入力をサニタイズ
- ✅ **HTTPS強制**: 本番環境で自動リダイレクト
- ✅ **設定分離**: 機密情報を config.js に分離
- ✅ **入力検証**: メールアドレス形式チェック

### 推奨される追加対策
- 🔜 サーバーサイド認証への移行
- 🔜 CSP ヘッダーの追加
- 🔜 2要素認証 (2FA)
- 🔜 CSRF トークン

詳細は [SECURITY.md](./SECURITY.md) を参照。

## 🐛 トラブルシューティング

### 管理画面にログインできない

```
❌ 設定エラー: config.js が見つかりません
```

**解決策**: 
1. `setup-security.html` を開く
2. 認証情報を入力して config.js を生成
3. ページをリロード

### セッションがすぐに切れる

**原因**: デフォルトのタイムアウトは1時間

**解決策**: `config.js` で延長

```javascript
SECURITY: {
  sessionTimeout: 7200000, // 2時間
}
```

### GASにデータが保存されない

**チェックポイント**:
1. GAS_URL が正しく設定されているか
2. GASのデプロイ設定で「誰でも」アクセス可能になっているか
3. ブラウザのコンソールでエラーを確認
